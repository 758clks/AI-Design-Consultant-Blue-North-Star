AI设计顾问 Chrome 扩展安装说明

推荐安装方式

解压 AI设计顾问-Chrome扩展.zip。

打开 Chrome，在地址栏输入 chrome://extensions/。

右上角打开“开发者模式”。

点击“加载已解压的扩展程序”。
选择解压后的 AI设计顾问-Chrome扩展 文件包。

安装后点击浏览器右上角扩展图标，或在侧边栏里打开“AI设计顾问”。

更新扩展

重新运行 npm run extension。

回到 chrome://extensions/。

找到“AI设计顾问”，点击刷新按钮。

如果刷新后仍异常，先移除旧扩展，再重新加载 AI设计顾问-Chrome扩展 文件包。

填写 Key  填写 说明

打开扩展后点击“填写 Key”或“更换 Key”。

通常选择模型厂商并填写 API Key 即可。

OpenAI 默认使用 gpt-5.2。

自定义厂商需要填写该厂商的 OpenAI 兼容 Base URL 和视觉模型名。

使用环境说明

扩展程序已内置前端页面，不依赖本机 localhost，换电脑或换网络后也可以打开。

但 AI 视觉识别需要模型厂商接口可访问，并且该厂商允许 Chrome 扩展直连。 

如果网页端可以识别、扩展端失败，通常是厂商接口不允许浏览器扩展直连，或扩展里保存的 Key/模型名和网页端后端配置不同。

常见问题

显示“AI 视觉识别失败”：检查 Key、模型名、额度和网络。

显示“无法直连当前模型厂商接口”：该厂商可能限制浏览器跨域请求，建议改用网页端或公网后端。

看不到侧边栏：确认 Chrome 版本支持 Side Panel，并重新加载扩展。

换了 zip 后没变化：需要在 chrome://extensions/ 里刷新扩展，或删除旧扩展后重新加载。
